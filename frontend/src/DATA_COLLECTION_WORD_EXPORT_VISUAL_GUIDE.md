## 🚨 Common Issues & Solutions

### Issue 1: Export Button Not Working
**Symptoms**: Click button, nothing happens
**Solution**: 
- Check browser console for errors
- Verify all required fields are filled
- Check internet connection (for library CDN)

### Issue 2: Missing Sections
**Symptoms**: Some sections don't appear in Word
**Solution**: 
- Check if data exists for that section
- Empty sections are automatically hidden
- This is expected behavior

### Issue 3: Wrong Data in Table
**Symptoms**: Numbers or text in wrong columns
**Solution**:
- Verify field mapping in DataCollection.tsx
- Check modal interface definitions
- Ensure data types match (number vs string)

### Issue 4: File Won't Open
**Symptoms**: Word says file is corrupted
**Solution**:
- Try re-exporting
- Check browser compatibility
- Clear browser cache

## 💡 Pro Tips

### For Best Results:

1. **Fill All Sections**:
   - More data = more comprehensive document
   - Use all modals: Material, Manpower, Equipment, Schedule, Consumables

2. **Use Consistent Naming**:
   - Material names should be consistent for grouping
   - Use standard units (Kgs, Pcs, m3)

3. **Add Notes**:
   - Use keterangan fields for important info
   - Notes appear in tables for reference

4. **Check Before Export**:
   - Review all data in "Lihat Detail"
   - Verify calculations
   - Then export

5. **Save Multiple Versions**:
   - Export at different project stages
   - Use Rev field to track versions
